# EmployeeManagementSystem

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 14.2.1.
This Employee portal helps a company manage their employee data by allowing an admin to create new employees in their system. In Version 1.01, an admin can create, read, update or delete employee data.
